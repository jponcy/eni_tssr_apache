Les balises block

Les balises blocks sont des balises qui prennent de base toute la largeur de l'écran et qui provoquent un "retour chariot" visuel sur le navigateur. L'élement block prend donc toute la ligne (par défaut).

Voici quelques balises blocks :

h[1-6]
titre de niveaux 1 à 6 ; 1 est le titre de la page (ou de l'article/section)
p
déclare un paragraphe
ul
déclare une liste non ordonnée (le type de puce pourra être personalisé en CSS, entre autre choses)
ol
déclare une liste ordonnée (le type de puce pourra être personalisé en CSS, entre autre choses)
dl (dt/dd)
liste de définitions (style dictionnaire/glossaire)
img
place une image
article/section/main/header/footer/...
balises structurantes
div
balise neutre (permet de placer une balise block sans aucun sens sémantique)

Les balises inline

Permettent de décrire ou d'ajouter des attributs sur une portion de code.
Une balise inline ne doit jamais contenir une balise de type block. 

          
Bonjour Jean, comment ça va ?

Principales balises inline :

a
lien (externe/interne/ancrage/ressource à télécharger/envoi email/...)
mark>strong>em
permettent de décrire un bout de phrase comme important
span
balise neutre (permet de placer une balise inline sans aucun sens sémantique)
